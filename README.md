# Habit Tracker App with Reward System

A gamified daily habit tracker Android app built with Kotlin.

## Features
- Track daily habits
- Earn points and XP for completing habits
- Level up system
- Streak tracking with bonus rewards
- Reward shop to redeem points
- Statistics dashboard
- Dark/Light theme support

## Tech Stack
- Kotlin
- Room Database
- MVVM Architecture
- Material Design Components
- Coroutines & LiveData

## Building the APK
The repository includes GitHub Actions workflow that automatically builds the APK.
Check the Actions tab after pushing code to download the compiled APK.
